# Banking-Risk-Analysis
This project focuses on analyzing banking data to reduce lending risk and support better decision-making using data analytics and visualization.

 Overview

The dashboard helps identify whether a customer is likely to repay a loan based on their financial profile, enabling smarter loan approval decisions.

The dataset includes multiple related tables such as:
Client Banking
Banking Relationship
Gender
Investment Advisor
Time Period

These are connected using primary and foreign keys.

📊 Key Metrics
Total Clients: 3000
Total Loan: 4.38B
Total Deposit: 3.77B
Business Lending: 2.6B
Checking & Saving Accounts

🛠️ Tools Used
Python (EDA & Cleaning)
MySQL (Querying)
Power BI (Dashboard)

🔎 Key Steps
Data cleaning and preprocessing in Python
Exploratory Data Analysis (EDA)
Feature engineering (Income Band, Engagement Metrics, Fees)
SQL queries for data extraction
Interactive dashboard creation in Power BI

📈 Dashboard
Includes:
Home
Loan Analysis
Deposit Analysis
Summary Dashboard

🎯 Conclusion
The project demonstrates how data analytics + Power BI can help banks understand customer behavior and minimize financial risk.
